# ✿ StudyBloom — Personal Exam Planner

StudyBloom is a custom, browser-based study dashboard designed to help track university exam schedules, internal marks, and syllabus progress.

#🚀 Purpose
I built this tool to manage my **Semester 4 Electronics and Communication Engineering (ECE)** exams under the **KTU 2024 scheme**. It features a dynamic countdown, target SGPA calculator, and an interactive study timetable.

#🤖 AI Disclosure & Disclaimer
* **Built with AI:** This website was created with the assistance of AI (Claude/Gemini).
* **Potential Flaws:** As an AI-assisted project, there may be functional bugs, layout inconsistencies on certain devices, or logical errors in automated calculations.
* **Manual Verification:** Users should manually verify exam dates and mark calculations against official university notifications.

#🔒 Privacy
This app uses **Local Storage**. All data you enter stays on your own device and is not uploaded to any server or monitored by any third party.
